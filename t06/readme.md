# T06: Accés remot – Escriptori remot (RDP)

## Breu descripció
Aquesta tasca individual té com a objectiu crear **guies de suport gràfic** per a l’accés remot a equips d’usuari mitjançant **RDP**, facilitant el suport tècnic directe i la resolució d’incidències.

---

## Context
Com a consultors d’**EverPia**, cal donar suport remot tant a servidors Windows amb interfície gràfica com a equips d’usuari final. L’accés mitjançant RDP permet veure la pantalla de l’usuari i controlar ratolí i teclat de manera remota.

---

## Objectiu de la tasca
- Elaborar guies clares i entenedores per a futurs becaris.
- Realitzar una **Prova de Concepte (PoC)** amb màquines virtuals.
- Documentar la connexió remota via **RDP**:
  - Windows ↔ Windows
  - Linux (Zorin / GNOME) ↔ Windows

---

## Tecnologia utilitzada
- **RDP (Remote Desktop Protocol)**  
  Protocol estàndard per a escriptori remot en entorns Windows i compatible amb sistemes GNU/Linux moderns.
