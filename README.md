# Project4 _“Sobreviure en una empresa IT”_

## Introducción
Última fase del recorrido como **junior IT** dentro de EverPia. Este proyecto final sirve para demostrar madurez profesional, autonomía y capacidad técnica antes de dar el salto a un rol más independiente.

---

## Objetivos del Proyecto

- **Integrar conocimientos técnicos** en un contexto real.
- **Dominar procesos de backup** y restauración.
- **Desplegar servicios IT corporativos** (NFS, CUPS, SSH, acceso remoto).
- **Diseñar un prototipo corporativo + ecommerce** inicial en Figma.
- **Iniciar la gestión de versiones con Git**.
- **Mantener un Kanban actualizado**.
- **Documentar procesos con calidad profesional**.
- **Aplicar criterios de sostenibilidad y economía circular**.

---

## Retos Principales

### Bloque Técnico
- **Creación y restauración** de imágenes con Rescuezilla.
- **Estrategias de backup** para entornos Windows.
- **Despliegue de servidor NFS** (con y sin LDAP).
- **Implementación de servidor de impresión CUPS**.
- Evaluación y preparación de **soluciones de acceso remoto**.

### Bloque Creativo y Organizativo
- **Prototipado corporativo + ecommerce** en Figma.
- **Repositorio local Git** con primeros commits organizados.
- **Documentación técnica profesional**.
- **Gestión completa del proyecto mediante Kanban**.
- **Inclusión de perspectiva sostenible**.

---

## Entregables Finales (Carpeta Profesional)

- Imagen de restauración funcional + prueba documentada.
- Informe comparativo de soluciones de backup + propuesta final.
- Servidor **NFS** operativo y documentado.
- Servidor **CUPS** con pruebas realizadas.
- Informe completo de soluciones de **acceso remoto**.
- Prototipo **corporativo + ecommerce** inicial en Figma.
- Repositorio **Git** con commits coherentes.
- **Kanban** completo y actualizado.
- Reflexión sobre sostenibilidad y economía circular.

---

## Resumen Final
Este proyecto es la culminación del periodo de prácticas: una demostración de habilidades técnicas, organizativas y profesionales que marca el paso de estudiante a un perfil júnior capacitado para el entorno IT real.
